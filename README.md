# Citibike-ETL-Pipeline

```bash
docker compose run --workdir /usr/app dbt init citibike_project

```

This command initializes a new dbt project named `citibike_project` inside a Docker container. The `--workdir /usr/app` option sets the working directory inside the container to `/usr/app`, where the project will be created. Make sure you have Docker and Docker Compose installed to run this command successfully.
